# AI Usage

This project used AI coding assistants during the hackathon build.

## Scope

AI assistance was used for:

- code review and implementation support;
- Solidity, TypeScript, Python, and frontend iteration;
- test generation and debugging;
- documentation drafts;
- project positioning and demo-script planning.

## Human Responsibilities

The project owner reviewed the product direction, approved the final positioning, and is responsible for any hackathon submission, investor communication, deployment, and live demonstration.

## Verification

AI-generated or AI-assisted code was checked through:

- GitHub Actions CI;
- Foundry contract tests in CI;
- keeper TypeScript typecheck and Vitest;
- TypeScript SDK typecheck;
- Python SDK import smoke;
- live frontend and VPS service smoke checks.

No hackathon submission should be made from this repository without explicit owner approval.
